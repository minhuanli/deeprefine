# deeprefine

## Installation

1. Install pytorch

2. Install other dependencies
   
   ```shell
   mamba install -c conda-forge pdbfixer
   mamba install -c conda-forge mdtraj
   pip install healpy
   ```

3. Install `deeprefine`

   ```shell
   git clone https://github.com/minhuanli/deeprefine.git
   cd deeprefine
   pip install -e .
   ```
